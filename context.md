# PROJECT CONTEXT: Hey Lola (heylola.co.za)
**Role:** AI Concierge for South African Restaurants.
**Core Value:** Answers calls instantly, takes bookings, reduces staff interruptions, captures data.
**Target Audience:** Restaurant owners in Cape Town/SA. Skeptical, busy, hate missed calls.

## CRITICAL RULES
1. **No Avatars:** Do not use "blonde lady" avatars. Use product UI, waveforms, or abstract hospitality visuals.
2. **South African Context:** Currency is ZAR (R). Compliance is POPIA (not GDPR).
3. **Tone:** Premium hospitality, confident, concise. "Invisible Service."

## SITE STRUCTURE
1. **Hero:** "Every call answered. Every table booked." (Visual: Waveform/UI, not human).
2. **Pain Points:** Missed calls = lost revenue.
3. **Features:** Call transcription, booking integration, shift logs.
4. **How it Works:** 1. Forward calls -> 2. Lola answers -> 3. You get bookings.
5. **Trust:** POPIA compliant, secure data.
6. **CTA:** "Book a Demo" (Primary), "See it in action" (Secondary).
